# Street Fighter Game Bot - Spring 2025

## Setup Instructions

1. Install Python 3.6+ and pip
2. Install dependencies:
   pip install -r requirements.txt

3. Place the following files in this directory:
   - bot.py
   - trained_model.pkl
   - label_encoder.pkl

4. Download and extract the API zip (BizHawk + Street Fighter ROM)
   Link: <provided>

## Running the Bot

1. Open BizHawk (`EmuHawk.exe`)
2. Load the ROM: `Street Fighter II Turbo (U).smc`
3. Open Tool Box (Shift + T)
4. In this folder, run:
   python controller.py 1
5. Go to the emulator, select player and press the **Gyroscope Bot icon**
6. "Connected successfully!" should appear
